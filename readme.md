# Módulo Gofas GalaxPay Cartão para WHMCS

[GalaxPay](https://app.galaxpay.com.br/abrir-conta?affiliateHash=34c8f0bb) é um dos melhores sistemas de gestão de cobranças recorrentes e únicas.

O módulo de integração [**GalaxPay Cartão para WHMCS**](https://gofas.net/ggpc/) processa pagamentos à vista e parcelados, automatiza pagamentos recorrentes e ainda permite capturar pagamentos manualmente pelo admin do [WHMCS](https://www.whmcs.com/members/aff.php?aff=35929).

Utiliza o sistema nativo de checkout transparente do [WHMCS](https://www.whmcs.com/members/aff.php?aff=35929) mas não armazena o cartão de crédito dos clientes no seu servidor.

Conheça também os módulos de integração [GalaxPay Boleto para WHMCS](https://gofas.net/ggpb/) e [GalaxPay Pix para WHMCS](https://gofas.net/ggpp/).
